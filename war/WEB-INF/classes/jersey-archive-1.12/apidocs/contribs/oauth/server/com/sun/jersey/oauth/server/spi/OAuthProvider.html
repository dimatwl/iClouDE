Interface OAuthProvider

All Known Implementing Classes:
DefaultOAuthProvider
public interface OAuthProvider

Contract for a provider that supports managing OAuth tokens and consumer secrets. To add an OAuthProvider implementation, annotate the implementation class with &#64;Provider.

Author:
Martin Matula
