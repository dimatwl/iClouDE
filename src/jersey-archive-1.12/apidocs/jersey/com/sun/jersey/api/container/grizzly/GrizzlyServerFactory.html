Create a SelectorThread that registers an Adapter that in turn manages all root resource and provder classes found by searching the classes referenced in the java classath.

Parameters:
u - the URI to create the http server. The URI scheme must be equal to "http". The URI user information and host are ignored If the URI port is not present then port 80 will be used. The URI path will be set as the resources context root value, which must be an empty String or begin with a "/". The URI query and fragment components are ignored.
adapter - the Adapter
Returns:
the select thread, with the endpoint started

java.lang.IllegalArgumentException - if u is null or the URI path does not begin with a "/".
